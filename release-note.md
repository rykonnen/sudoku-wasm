Release Notes – Sudoku Web Game

Project: Sudoku Web Game
Author: Leona Mae Blancaflor
Repository: https://github.com/rykonnen/sudoku-wasm

Initial Release Date: November 28, 2025

Version 1.0 – Initial Release
Date: November 28, 2025
Highlights:

Developed the core Sudoku logic entirely in C.
Implemented a recursive backtracking algorithm for puzzle generation and solving.
Added puzzle uniqueness verification via hole-digging method.
Designed a basic UI using CLAY for layouting and Raylib for rendering and input.
- Enabled WebAssembly (WASM) compilation for browser execution.
- Local server and GitHub Pages deployment tested successfully.

Known Limitations:
- Minimalist UI due to CLAY and WASM constraints.
- Difficulty settings based solely on clue count.
- ouchscreen input and mobile-friendly features not fully supported.
